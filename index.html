CSS Relative Gridline Positioning, Nested

The following rows are a grid nested inside a parent grid. The parent includes the blue header and footer. The center section contains an 8-column nested grid that uses relative line numbers to position each row.

I got this idea from Mozilla's CSS Layout Cookbook. I combined their CSS grid "sticky footer" recipe with the "grid wrapper" recipe, here.
